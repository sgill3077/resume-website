A responsive personal portfolio and resume website built with HTML, CSS, and JavaScript.

---

## Features

- Responsive mobile-friendly layout
- Sticky navigation bar
- Mobile hamburger menu
- Dark mode with persistence
- Smooth section animations
- Resume download support (PDF/DOCX)
- Expandable project sections
- GitHub project integration
- Clean modern UI
- Optimized for desktop and mobile devices

## Local Development

Clone the repository:

```bash
git clone https://github.com/sgill3077/resume-website.git
```

Navigate into the project directory:

```bash
cd resume-website
```

Run locally using Python:

```bash
python3 -m http.server 8000
```

Open in browser:

```text
http://localhost:8000
```
